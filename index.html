¿Qué es el Diseño Web Responsivo?

El diseño web responsivo permite que los sitios web se ajusten correctamente a cualquier tipo de dispositivo, desde teléfonos móviles hasta computadoras de escritorio.

Viewport

El viewport es el área visible de una página web y es crucial para asegurarse de que un sitio se vea bien en diferentes tamaños de pantalla. Para configurarlo, se usa la metaetiqueta viewport.

Media Queries

Las media queries permiten aplicar estilos CSS según las características del dispositivo, como el ancho de la pantalla. Con ellas, se puede modificar el diseño en función del tamaño de la ventana del navegador.

Grid System

El sistema de cuadrículas de Bootstrap es fundamental para construir diseños flexibles y responsivos. Permite dividir el contenido en columnas que se ajustan según el tamaño de la pantalla.
